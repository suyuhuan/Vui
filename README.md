# Vui 一个Vue UI组件库
#### 作者 SUYUHUAN
## 介绍
 这是一个使用vue 过程中做的一个UI框架
## 开始使用
1. 安装
使用本框架时，请在css中开启 border-box;
```
  *,*::before,*::after{box-sizing: border-box;}
```
还需要设置默认颜色变量s
```
:root{
            --button-height:32px;
            --font-size:14px;
            --button-bg:white;
            --button-active-bg:#eee;
            --border-radius: 4px;
            --color:#999;
            --border-color:#999;
            --border-color-hover:#666;
        }
```

IE8 及以上浏览器都支持样式
